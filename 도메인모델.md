### 관리자 (Manager)

#### 속성
- `managerId`: 관리자 ID - ID
- `name`: 이름
- `passwordHash`: 비밀번호
- `institution`: 기관
- 'birthDate': 생년월일

#### 행위
- `static create()`: 관리자 생성: ID, nickname, password, passwordEncoder
#### 규칙
- 비밀번호는 해시를 만들어서 저장한다.


### 직원 (Worker)

#### 속성

- `workerId`: 직원 ID - ID
- `name`: 이름
- `occupation`: 직군
- `department`: 부서
- `birthDate`: 생년월일
- `hireDate`: 입사일
- `gender`: 성별
- `field` : 공정
- `workType`: 근무형태

#### 행위


#### 규칙
- 직원 ID는 고유해야 한다.
- 직원은 여러개의 사후관리소견서를 가질 수 있다.
