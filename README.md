# 사후관리소견서 관리 서비스

## 소개

### 사후관리소견서 관리 서비스는 직원의 사후관리소견서를 효율적으로 관리하고 추적할 수 있는 웹 기반 애플리케이션입니다. 이 서비스는 관리자가 직원의 상태를 지속적으로 모니터링하고, 필요한 조치를 신속하게 취할 수 있도록 돕습니다.

### 주요기능

1. 사후관리소견서 작성 및 편집
   - 관리자는 직원의 사후관리소견서를 작성하고, 필요에 따라 수정할 수 있습니다.

2. 사후관리소견서 목록 조회
    - 작성된 사후관리소견서의 목록을 조회하고, 특정 직원의 소견서를 검색할 수 있습니다.

3. 보건상담 작성 및 관리
    - 직원의 보건상담 내용을 기록하고, 상담 이력을 관리할 수 있습니다.


### 개발 환경

- 화면 구현 : thymeleaf
- 백엔드 : Spring Boot
- 데이터베이스 : MySQL
- 빌드 도구 : Gradle
- 버전 관리 : Git
- 배포 : AWS EC2
